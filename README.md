# SubmeetDB
sviluppo di un software per l’esame di Ingegneria del Software, con funzionalità di registrazione utenti, creazione e iscrizione a conferenze, caricamento e revisione articoli; progettazione di database e interfaccia utente in Java, con documentazione ODD, RAD e SDD.


1) Crea il Database: Assicurati di avere un database chiamato "submeetdb"

2) Configura le Credenziali: Apri il file DBMSBoundary.java e modifica le seguenti costanti con i parametri del tuo server locale:
private static final String URL = "jdbc:mariadb://localhost:3306/submeetdb";
private static final String USER = "IL_TUO_UTENTE"; 
private static final String PASSWORD = "LA_TUA_PASSWORD";
